## 安裝前你需要
安裝 Python >= 3.6 (建議使用 3.12.6 或 3.13，因為已經測試過)

建立一個**單獨**的資料夾來運行遊戲（如果在提示框中點擊“是”，它將刪除資料夾中的所有檔案）
